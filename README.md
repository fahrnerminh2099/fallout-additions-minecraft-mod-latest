### How to Craft Power Armor

1. Mine iron, steel, copper, and rare materials
2. Craft **Power Armor Station**
3. Place station and right-click
4. Start with **T-45 Frame**
5. Add armor pieces (helmet, chest, legs, arms)
6. Insert **Fusion Core** (crafted from copper + redstone)
7. Right-click frame to equip

### How to Get Nuka-Cola

1. Find Nuka-Cola machines in ruined buildings
2. Right-click machine to receive 1-3 bottles
3. Or craft:
   - **Nuka-Cola** (4 sugar + 2 glass + 1 iron)
   - **Nuka-Cola Quantum** (Nuka + glowstone)
4. Drink for speed/hunger buffs
5. Craft **Mixology Station** for Nuka recipes

### How to Survive Radiation

1. Watch Geiger counter UI (top left)
2. Avoid glowing blocks and certain biomes
3. If radiating:
   - Use **Rad-X** (temporary protection)
   - Use **RadAway** (removes accumulated radiation)
4. Build **Decontamination Arch** for base
5. Radiation causes hunger, weakness, and death at max

### How to Find Vaults

1. Explore caves below Y=30
2. Look for vault doors in cave walls
3. Requires **Vault-Tec Keycard** (found in ruined buildings)
4. Enter vault
5. Clear enemies (feral ghouls, radroaches)
6. Loot unique items and terminal lore

### How to Use VATS

1. Equip any weapon
2. Press **"V"** key (default)
3. Time slows down
4. Click on enemy to target specific body parts
5. Each part has hit chance
6. Press again to execute shots
7. VATS costs hunger to use

### How to Craft Energy Weapons

1. Build **Weapon Workbench**
2. Gather advanced materials: steel, copper, refined iron
3. Craft **Laser Pistol** (basic)
4. Upgrade to **Laser Rifle** at workbench
5. Find **Plasma** components from super mutants
6. Craft **Plasma Rifle** for higher damage
